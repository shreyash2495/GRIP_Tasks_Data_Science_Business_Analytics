Prediction using the Decision Tree Algorithm
Creating the Decision Tree classifier and visualize it graphically.
Dataset contains 6 variables
1. ID - Unique ID
2. Sepallengthcm - Sepal Length in cm.
3. Sepalwidthcm - Sepal Width in cm.
4. Petallengthcm - Petal Length in cm.
5. Petalwidthcm - Petal Width in cm.
6. Species - Also known as Class which has 3 unique values  
                                           -- Iris Setosa
                                           -- Iris Versicolour
                                           -- Iris Virginica
